# AEGIS — AI Enterprise Gateway & Intelligence System

## Project Overview
Enterprise AI Support Platform — 기업이 LLM을 안전하고 효율적으로 사용할 수 있게 해주는 셀프호스팅 가능한 AI 게이트웨이 플랫폼

## Tech Stack
- **Language**: TypeScript
- **Backend**: NestJS
- **Frontend**: Next.js
- **Database**: PostgreSQL + pgvector
- **Cache/Queue**: Redis + BullMQ
- **Observability**: Langfuse (self-hosted)

## SDD Workflow
This project uses spec-kit-skills for Specification-Driven Development:
1. `/domain-extend` — Custom domain modules (ai-gateway, token-budget, prompt-guard)
2. `/smart-sdd init` — Project initialization (Proposal Mode)
3. `/smart-sdd add` — Feature briefing
4. `/smart-sdd pipeline` — specify → plan → tasks → implement → verify

## Rules
- All specs and plans must be generated before implementation
- Feature-by-Feature session strategy (avoid context saturation)
- Log any spec-kit-skills issues immediately to `docs/skill-feedback.md`
- Case study documentation in `docs/case-study.md` must be updated at each phase
